Installation
************

Supported Platforms
###################
-  Linux
-  Windows

Installing with pip
###################
Simply install with pip.

::

    pip3 install monitorcontrol

Manual Installation
###################
Clone (or download) and install the package.

::

    git clone https://github.com/newAM/monitorcontrol.git
    cd monitorcontrol
    python3 setup.py install
